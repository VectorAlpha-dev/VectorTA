# Rust Backtester Web Interface

This folder contains the Astro powered website for the Rust Backtester project.
The site provides a lightweight dashboard that visualizes indicator data and
strategy output generated by the core backtester. It can also load the
compiled WebAssembly version of the library to run calculations directly in the
browser.

Place CSV files exported from the Rust backtester inside the `static/` directory
to make them available for charts and examples. After running `npm run build`
the resulting `dist/` folder can be served as a static site alongside your
backtesting binaries.

## ðŸ§ž Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
